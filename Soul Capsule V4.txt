# SoulCapsule v4 — Full Codebase

> A Personal Multimodal Memory Vault. Simple. Beautiful. Alive.

---

## What's New in v4

- **`core.py`** — all logic extracted to a clean shared module; CLI and UI are thin wrappers
- **`IndexIDMap2`** — real vector deletion without full index rebuild
- **Media import always on** — `--no-import` flag to opt out; vault is self-contained by default
- **Soft delete / versioning** — memories are marked `deleted_at`, never hard-removed; recoverable
- **Duplicate detection** — cosine similarity check (>0.92) warns before saving a near-duplicate
- **Multimodal query** — search with an image or audio clip, not just text
- **Entity extraction** — spaCy NER auto-enriches tags on add
- **Lazy model loading** — Whisper and CLIP load only when needed (no penalty for text-only use)
- **`export` command** — `soulcapsule export --format json` or `--format zip` for full vault backup
- **Auto-backup** — configurable periodic backup to `soulcapsule_backups/`
- **`.env` support** — OpenAI key and config via environment
- **`typer` + `rich` CLI** — replaces argparse; beautiful output
- **Streamlit UI upgrades** — thumbnails in results, bulk delete, memory timeline heatmap, resonance scores with explanation, voice recording, dark-mode polish

```python
"""
SoulCapsule v4 — Core Logic
All database, embedding, FAISS, media, and business logic lives here.
CLI and UI import from this module only.
"""

from __future__ import annotations

import hashlib
import json
import os
import pickle
import shutil
import sqlite3
import tempfile
import uuid
import zipfile
from datetime import datetime, timezone
from pathlib import Path
from typing import Optional

import numpy as np
from dotenv import load_dotenv
from rich.console import Console

load_dotenv()
console = Console()

# ---------------------------------------------------------------------------
# Configuration
# ---------------------------------------------------------------------------

CONFIG = {
    "db_path":      Path(os.getenv("SC_DB_PATH",      "soulcapsule.db")),
    "faiss_path":   Path(os.getenv("SC_FAISS_PATH",   "soulcapsule.faiss")),
    "idmap_path":   Path(os.getenv("SC_IDMAP_PATH",   "soulcapsule_idmap.pkl")),
    "media_dir":    Path(os.getenv("SC_MEDIA_DIR",    "soulcapsule_media")),
    "backup_dir":   Path(os.getenv("SC_BACKUP_DIR",   "soulcapsule_backups")),
    "embed_dim":    384,          # all-MiniLM-L6-v2 output size
    "dup_threshold": 0.92,        # cosine similarity above which we warn about duplicates
    "openai_key":   os.getenv("OPENAI_API_KEY", ""),
}

MOOD_VALENCE: dict[str, float] = {
    "joy":       1.0,
    "love":      0.9,
    "excitement":0.8,
    "calm":      0.4,
    "nostalgia": 0.1,
    "neutral":   0.0,
    "sadness":  -0.6,
    "fear":     -0.7,
    "anger":    -0.9,
}

VALID_MOODS = list(MOOD_VALENCE.keys())

# ---------------------------------------------------------------------------
# Lazy model registry
# ---------------------------------------------------------------------------

_MODELS: dict = {}


def _get_text_embedder():
    if "text" not in _MODELS:
        from sentence_transformers import SentenceTransformer
        _MODELS["text"] = SentenceTransformer("all-MiniLM-L6-v2")
    return _MODELS["text"]


def _get_clip():
    if "clip" not in _MODELS:
        try:
            import clip
            import torch
            device = "cuda" if torch.cuda.is_available() else "cpu"
            model, preprocess = clip.load("ViT-B/32", device=device)
            _MODELS["clip"] = (model, preprocess, device)
        except Exception as e:
            console.print(f"[yellow]CLIP unavailable — image embedding disabled: {e}[/yellow]")
            _MODELS["clip"] = None
    return _MODELS["clip"]


def _get_whisper():
    if "whisper" not in _MODELS:
        try:
            import whisper
            _MODELS["whisper"] = whisper.load_model("base")
        except Exception as e:
            console.print(f"[yellow]Whisper unavailable — audio transcription disabled: {e}[/yellow]")
            _MODELS["whisper"] = None
    return _MODELS["whisper"]


def _get_spacy():
    if "spacy" not in _MODELS:
        try:
            import spacy
            _MODELS["spacy"] = spacy.load("en_core_web_sm")
        except Exception as e:
            console.print(f"[yellow]spaCy unavailable — entity extraction disabled: {e}[/yellow]")
            _MODELS["spacy"] = None
    return _MODELS["spacy"]


# ---------------------------------------------------------------------------
# Database
# ---------------------------------------------------------------------------

def init_db() -> None:
    """Create tables if they don't exist. Safe to call multiple times."""
    with _db() as conn:
        conn.executescript("""
            CREATE TABLE IF NOT EXISTS memories (
                id          TEXT PRIMARY KEY,
                faiss_id    INTEGER UNIQUE NOT NULL,
                text        TEXT NOT NULL,
                mood        TEXT NOT NULL DEFAULT 'neutral',
                tags        TEXT,
                intensity   REAL NOT NULL DEFAULT 1.0,
                image_path  TEXT,
                audio_path  TEXT,
                audio_transcript TEXT,
                embedding   BLOB,
                created_at  TEXT NOT NULL,
                deleted_at  TEXT
            );

            CREATE TABLE IF NOT EXISTS meta (
                key   TEXT PRIMARY KEY,
                value TEXT
            );
        """)
        # Seed FAISS counter if not present
        conn.execute(
            "INSERT OR IGNORE INTO meta(key, value) VALUES ('faiss_counter', '0')"
        )


def _db(path: Optional[Path] = None) -> sqlite3.Connection:
    conn = sqlite3.connect(path or CONFIG["db_path"])
    conn.row_factory = sqlite3.Row
    return conn


def _next_faiss_id(conn: sqlite3.Connection) -> int:
    row = conn.execute("SELECT value FROM meta WHERE key='faiss_counter'").fetchone()
    current = int(row["value"])
    conn.execute(
        "UPDATE meta SET value=? WHERE key='faiss_counter'", (str(current + 1),)
    )
    return current


# ---------------------------------------------------------------------------
# FAISS index  (IndexIDMap2 — supports real vector removal)
# ---------------------------------------------------------------------------

def _load_index():
    import faiss
    path = CONFIG["faiss_path"]
    if path.exists():
        index = faiss.read_index(str(path))
        idmap = pickle.loads(CONFIG["idmap_path"].read_bytes()) if CONFIG["idmap_path"].exists() else {}
        return index, idmap
    return _new_index()


def _new_index():
    import faiss
    flat = faiss.IndexFlatL2(CONFIG["embed_dim"])
    index = faiss.IndexIDMap2(flat)
    return index, {}


def _save_index(index, idmap: dict) -> None:
    import faiss
    faiss.write_index(index, str(CONFIG["faiss_path"]))
    CONFIG["idmap_path"].write_bytes(pickle.dumps(idmap))


def rebuild_index() -> None:
    """Reconstruct FAISS IndexIDMap2 from embeddings stored in SQLite."""
    import faiss
    index, idmap = _new_index()
    with _db() as conn:
        rows = conn.execute(
            "SELECT id, faiss_id, embedding FROM memories WHERE deleted_at IS NULL AND embedding IS NOT NULL"
        ).fetchall()
    for row in rows:
        vec = np.frombuffer(row["embedding"], dtype=np.float32).reshape(1, -1)
        fid = np.array([row["faiss_id"]], dtype=np.int64)
        index.add_with_ids(vec, fid)
        idmap[row["id"]] = row["faiss_id"]
    _save_index(index, idmap)
    console.print(f"[green]Index rebuilt — {len(rows)} vectors loaded.[/green]")


def _auto_rebuild_if_needed() -> None:
    if not CONFIG["faiss_path"].exists():
        with _db() as conn:
            count = conn.execute(
                "SELECT COUNT(*) as n FROM memories WHERE deleted_at IS NULL"
            ).fetchone()["n"]
        if count > 0:
            console.print("[yellow]Index missing — rebuilding from database…[/yellow]")
            rebuild_index()


# ---------------------------------------------------------------------------
# Embeddings
# ---------------------------------------------------------------------------

def _embed_text(text: str) -> np.ndarray:
    model = _get_text_embedder()
    vec = model.encode([text], normalize_embeddings=True)[0]
    return vec.astype(np.float32)


def _embed_image(path: str) -> Optional[np.ndarray]:
    result = _get_clip()
    if result is None:
        return None
    try:
        import clip
        import torch
        from PIL import Image
        model, preprocess, device = result
        img = preprocess(Image.open(path)).unsqueeze(0).to(device)
        with torch.no_grad():
            feat = model.encode_image(img)
            feat = feat / feat.norm(dim=-1, keepdim=True)
        raw = feat.cpu().numpy()[0]
        # Project CLIP 512-d → 384-d via mean pooling of overlapping windows
        target = CONFIG["embed_dim"]
        if raw.shape[0] != target:
            indices = np.linspace(0, raw.shape[0] - 1, target).astype(int)
            raw = raw[indices]
        return raw.astype(np.float32)
    except Exception as e:
        console.print(f"[yellow]Image embedding failed: {e}[/yellow]")
        return None


def _embed_audio(path: str) -> tuple[Optional[np.ndarray], Optional[str]]:
    model = _get_whisper()
    if model is None:
        return None, None
    try:
        result = model.transcribe(path)
        transcript = result.get("text", "").strip()
        if not transcript:
            return None, None
        return _embed_text(transcript), transcript
    except Exception as e:
        console.print(f"[yellow]Audio transcription failed: {e}[/yellow]")
        return None, None


def _fuse_embeddings(*vecs: Optional[np.ndarray]) -> np.ndarray:
    """Average non-None embeddings into a single normalised vector."""
    valid = [v for v in vecs if v is not None]
    if not valid:
        raise ValueError("No embeddings to fuse.")
    combined = np.mean(np.stack(valid), axis=0)
    norm = np.linalg.norm(combined)
    if norm > 0:
        combined /= norm
    return combined.astype(np.float32)


# ---------------------------------------------------------------------------
# Entity extraction (spaCy NER → auto-tags)
# ---------------------------------------------------------------------------

def _extract_entities(text: str) -> list[str]:
    nlp = _get_spacy()
    if nlp is None:
        return []
    doc = nlp(text)
    entities = [
        ent.text.lower()
        for ent in doc.ents
        if ent.label_ in {"PERSON", "GPE", "LOC", "ORG", "EVENT", "PRODUCT", "WORK_OF_ART"}
    ]
    return list(dict.fromkeys(entities))  # deduplicate, preserve order


# ---------------------------------------------------------------------------
# Media import
# ---------------------------------------------------------------------------

def _import_media(path: str) -> str:
    """Copy file into content-addressed vault. Returns managed path."""
    src = Path(path)
    digest = hashlib.sha256(src.read_bytes()).hexdigest()
    dest_dir = CONFIG["media_dir"] / digest[:2]
    dest_dir.mkdir(parents=True, exist_ok=True)
    dest = dest_dir / (digest + src.suffix)
    if not dest.exists():
        shutil.copy2(src, dest)
    return str(dest)


# ---------------------------------------------------------------------------
# Duplicate detection
# ---------------------------------------------------------------------------

def _find_duplicate(embedding: np.ndarray) -> Optional[dict]:
    """
    Search for an existing memory with cosine similarity > threshold.
    Returns the closest match (as dict) or None.
    """
    index, idmap = _load_index()
    if index.ntotal == 0:
        return None
    vec = embedding.reshape(1, -1)
    distances, faiss_ids = index.search(vec, 1)
    if faiss_ids[0][0] == -1:
        return None
    # L2 distance → cosine similarity (vectors are unit-normalised)
    # cos_sim = 1 - (d² / 2)  for unit vectors
    l2_sq = float(distances[0][0])
    cos_sim = 1.0 - l2_sq / 2.0
    if cos_sim < CONFIG["dup_threshold"]:
        return None
    fid = int(faiss_ids[0][0])
    # Reverse lookup: faiss_id → memory uuid
    reverse = {v: k for k, v in idmap.items()}
    mem_id = reverse.get(fid)
    if mem_id is None:
        return None
    with _db() as conn:
        row = conn.execute(
            "SELECT id, text, mood, created_at FROM memories WHERE id=?", (mem_id,)
        ).fetchone()
    if row:
        return {**dict(row), "similarity": round(cos_sim, 3)}
    return None


# ---------------------------------------------------------------------------
# Core CRUD
# ---------------------------------------------------------------------------

def add_memory(
    text: str,
    image_path: Optional[str] = None,
    audio_path: Optional[str] = None,
    mood: str = "neutral",
    tags: Optional[str] = None,
    intensity: float = 1.0,
    import_media: bool = True,
) -> tuple[str, Optional[dict]]:
    """
    Save a new memory.
    Returns (memory_id, duplicate_warning_dict | None).
    """
    init_db()
    _auto_rebuild_if_needed()

    # --- Media import ---
    if image_path and import_media:
        image_path = _import_media(image_path)
    if audio_path and import_media:
        audio_path = _import_media(audio_path)

    # --- Embeddings ---
    text_vec = _embed_text(text)
    img_vec  = _embed_image(image_path) if image_path else None
    aud_vec, transcript = _embed_audio(audio_path) if audio_path else (None, None)
    fused = _fuse_embeddings(text_vec, img_vec, aud_vec)

    # --- Duplicate check ---
    duplicate = _find_duplicate(fused)

    # --- Entity enrichment ---
    entities = _extract_entities(text)
    existing_tags = [t.strip() for t in tags.split(",")] if tags else []
    all_tags = existing_tags + [e for e in entities if e not in existing_tags]
    tags_str = ", ".join(all_tags) if all_tags else None

    # --- Persist ---
    mem_id = str(uuid.uuid4())
    now = datetime.now(timezone.utc).isoformat()

    with _db() as conn:
        fid = _next_faiss_id(conn)
        conn.execute(
            """INSERT INTO memories
               (id, faiss_id, text, mood, tags, intensity,
                image_path, audio_path, audio_transcript,
                embedding, created_at, deleted_at)
               VALUES (?,?,?,?,?,?,?,?,?,?,?,NULL)""",
            (
                mem_id, fid, text, mood, tags_str, intensity,
                image_path, audio_path, transcript,
                fused.tobytes(), now,
            ),
        )

    index, idmap = _load_index()
    vec = fused.reshape(1, -1)
    fid_arr = np.array([fid], dtype=np.int64)
    index.add_with_ids(vec, fid_arr)
    idmap[mem_id] = fid
    _save_index(index, idmap)

    return mem_id, duplicate


def search_memories(
    query: str,
    k: int = 5,
    mood_filter: Optional[str] = None,
    query_image_path: Optional[str] = None,
    query_audio_path: Optional[str] = None,
) -> list[dict]:
    """
    Search memories by text (and optionally image/audio) query.
    Returns ranked list of memory dicts with 'resonance' score.
    """
    _auto_rebuild_if_needed()

    text_vec = _embed_text(query)
    img_vec  = _embed_image(query_image_path) if query_image_path else None
    aud_vec, _ = _embed_audio(query_audio_path) if query_audio_path else (None, None)
    query_vec = _fuse_embeddings(text_vec, img_vec, aud_vec)

    index, idmap = _load_index()
    if index.ntotal == 0:
        return []

    fetch_k = min(k * 5, index.ntotal)  # over-fetch for post-filtering
    distances, faiss_ids = index.search(query_vec.reshape(1, -1), fetch_k)

    reverse = {v: k for k, v in idmap.items()}
    results = []

    with _db() as conn:
        for dist, fid in zip(distances[0], faiss_ids[0]):
            if fid == -1:
                continue
            mem_id = reverse.get(int(fid))
            if not mem_id:
                continue
            row = conn.execute(
                "SELECT * FROM memories WHERE id=? AND deleted_at IS NULL", (mem_id,)
            ).fetchone()
            if not row:
                continue
            row = dict(row)
            row.pop("embedding", None)
            if mood_filter and row["mood"] != mood_filter:
                continue
            cos_sim = 1.0 - float(dist) / 2.0
            valence = MOOD_VALENCE.get(row["mood"], 0.0)
            row["resonance"] = round(cos_sim * row["intensity"] * (1 + 0.2 * abs(valence)), 3)
            row["resonance_explanation"] = (
                f"Semantic match {cos_sim:.2f} × intensity {row['intensity']:.1f} "
                f"× mood weight {1 + 0.2 * abs(valence):.2f}"
            )
            results.append(row)
            if len(results) >= k:
                break

    results.sort(key=lambda r: r["resonance"], reverse=True)
    return results


def delete_memory(id_prefix: str, hard: bool = False) -> bool:
    """
    Soft-delete (default) or hard-delete a memory by full ID or prefix.
    Soft delete marks deleted_at and removes vector from FAISS.
    Hard delete also purges the DB row.
    Returns True on success.
    """
    with _db() as conn:
        rows = conn.execute(
            "SELECT id, faiss_id FROM memories WHERE id LIKE ? AND deleted_at IS NULL",
            (id_prefix + "%",),
        ).fetchall()

    if len(rows) != 1:
        return False

    mem_id = rows[0]["id"]
    fid    = rows[0]["faiss_id"]

    # Remove from FAISS (IndexIDMap2 supports this natively)
    index, idmap = _load_index()
    fid_arr = np.array([fid], dtype=np.int64)
    index.remove_ids(fid_arr)
    idmap.pop(mem_id, None)
    _save_index(index, idmap)

    now = datetime.now(timezone.utc).isoformat()
    with _db() as conn:
        if hard:
            conn.execute("DELETE FROM memories WHERE id=?", (mem_id,))
        else:
            conn.execute("UPDATE memories SET deleted_at=? WHERE id=?", (now, mem_id))

    return True


def restore_memory(id_prefix: str) -> bool:
    """Restore a soft-deleted memory back into the active index."""
    with _db() as conn:
        rows = conn.execute(
            "SELECT id, faiss_id, embedding FROM memories WHERE id LIKE ? AND deleted_at IS NOT NULL",
            (id_prefix + "%",),
        ).fetchall()

    if len(rows) != 1:
        return False

    row = rows[0]
    mem_id = row["id"]
    fid    = row["faiss_id"]
    vec    = np.frombuffer(row["embedding"], dtype=np.float32).reshape(1, -1)

    index, idmap = _load_index()
    index.add_with_ids(vec, np.array([fid], dtype=np.int64))
    idmap[mem_id] = fid
    _save_index(index, idmap)

    with _db() as conn:
        conn.execute("UPDATE memories SET deleted_at=NULL WHERE id=?", (mem_id,))

    return True


def get_all_memories(include_deleted: bool = False) -> list[dict]:
    with _db() as conn:
        sql = "SELECT * FROM memories" if include_deleted else \
              "SELECT * FROM memories WHERE deleted_at IS NULL"
        rows = conn.execute(sql + " ORDER BY created_at DESC").fetchall()
    result = []
    for row in rows:
        d = dict(row)
        d.pop("embedding", None)
        result.append(d)
    return result


def surprise_me() -> Optional[dict]:
    """Surface a forgotten memory (weighted toward older, less-viewed)."""
    with _db() as conn:
        rows = conn.execute(
            "SELECT * FROM memories WHERE deleted_at IS NULL ORDER BY created_at ASC"
        ).fetchall()
    if not rows:
        return None
    # Weight: older memories get higher probability
    weights = np.exp(-np.linspace(0, 3, len(rows)))
    weights /= weights.sum()
    idx = np.random.choice(len(rows), p=weights)
    row = dict(rows[idx])
    row.pop("embedding", None)
    return row


def mood_drift(n_buckets: int = 30) -> list[dict]:
    """Return bucketed mood-valence data for timeline plotting."""
    with _db() as conn:
        rows = conn.execute(
            "SELECT mood, intensity, created_at FROM memories "
            "WHERE deleted_at IS NULL ORDER BY created_at ASC"
        ).fetchall()
    if not rows:
        return []

    min_ts = datetime.fromisoformat(rows[0]["created_at"])
    max_ts = datetime.fromisoformat(rows[-1]["created_at"])
    span   = max((max_ts - min_ts).total_seconds(), 1.0)
    buckets: list[list[float]] = [[] for _ in range(n_buckets)]

    for row in rows:
        dt  = datetime.fromisoformat(row["created_at"])
        pos = int((dt - min_ts).total_seconds() / span * (n_buckets - 1))
        val = MOOD_VALENCE.get(row["mood"], 0.0) * row["intensity"]
        buckets[pos].append(val)

    result = []
    for i, bucket in enumerate(buckets):
        ts = min_ts.timestamp() + i / (n_buckets - 1) * span
        result.append({
            "timestamp": datetime.fromtimestamp(ts).isoformat(),
            "valence":   round(sum(bucket) / len(bucket), 3) if bucket else None,
            "count":     len(bucket),
        })
    return result

# Export & Backup

def export_vault(fmt: str = "json", dest: Optional[str] = None) -> str:
    """
    Export the vault.
    fmt='json'  → single JSON file with all memories (no media blobs).
    fmt='zip'   → JSON + all media files in a zip archive.
    Returns path to output file.
    """
    memories = get_all_memories()
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    dest_path = Path(dest) if dest else Path(f"soulcapsule_export_{timestamp}")

    if fmt == "json":
        out = dest_path.with_suffix(".json")
        out.write_text(json.dumps(memories, indent=2, default=str))
        return str(out)

    elif fmt == "zip":
        out = dest_path.with_suffix(".zip")
        with zipfile.ZipFile(out, "w", zipfile.ZIP_DEFLATED) as zf:
            zf.writestr("memories.json", json.dumps(memories, indent=2, default=str))
            for mem in memories:
                for field in ("image_path", "audio_path"):
                    p = mem.get(field)
                    if p and Path(p).exists():
                        zf.write(p, arcname=f"media/{Path(p).name}")
        return str(out)

    raise ValueError(f"Unknown format: {fmt!r}. Use 'json' or 'zip'.")


def auto_backup() -> str:
    """Create a timestamped zip backup in soulcapsule_backups/."""
    CONFIG["backup_dir"].mkdir(parents=True, exist_ok=True)
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    dest = CONFIG["backup_dir"] / f"backup_{timestamp}"
    return export_vault("zip", str(dest))
```

```python
"""
SoulCapsule v4 — Typer CLI
Run: python cli.py --help
"""

from __future__ import annotations

from typing import Optional
import typer
from rich.console import Console
from rich.table import Table
from rich import box

from core import (
    init_db,
    add_memory,
    search_memories,
    delete_memory,
    restore_memory,
    rebuild_index,
    surprise_me,
    mood_drift,
    export_vault,
    auto_backup,
    get_all_memories,
    VALID_MOODS,
)

app     = typer.Typer(help="SoulCapsule v4 — Your personal multimodal memory vault.")
console = Console()


# ---------------------------------------------------------------------------
# add
# ---------------------------------------------------------------------------

@app.command()
def add(
    text: str = typer.Argument(..., help="Memory text"),
    image: Optional[str]  = typer.Option(None,  "--image",     help="Path to image file"),
    audio: Optional[str]  = typer.Option(None,  "--audio",     help="Path to audio file"),
    mood:  str            = typer.Option("neutral", "--mood",   help=f"One of: {', '.join(VALID_MOODS)}"),
    tags:  Optional[str]  = typer.Option(None,  "--tags",      help="Comma-separated tags"),
    intensity: float      = typer.Option(1.0,   "--intensity", min=0.1, max=2.0, help="Emotional intensity"),
    no_import: bool       = typer.Option(False, "--no-import", help="Skip copying media into vault"),
):
    """Capture a memory."""
    init_db()
    if mood not in VALID_MOODS:
        console.print(f"[red]Invalid mood. Choose from: {', '.join(VALID_MOODS)}[/red]")
        raise typer.Exit(1)

    with console.status("Embedding and saving…"):
        mem_id, duplicate = add_memory(
            text=text,
            image_path=image,
            audio_path=audio,
            mood=mood,
            tags=tags,
            intensity=intensity,
            import_media=not no_import,
        )

    if duplicate:
        console.print(
            f"[yellow]⚠ Possible duplicate detected (similarity {duplicate['similarity']}):[/yellow]\n"
            f"  ID: {duplicate['id'][:8]}  |  {duplicate['created_at'][:10]}  |  {duplicate['text'][:80]}"
        )
        if not typer.confirm("Save anyway?"):
            delete_memory(mem_id, hard=True)
            console.print("[dim]Memory discarded.[/dim]")
            return

    console.print(f"[green]✦ Memory saved — ID: {mem_id[:8]}[/green]")

@app.command()
def mood_drift_cmd(
    buckets: int = typer.Option(30, "--buckets", "-b", help="Timeline granularity"),
):
    """Print an emotional arc sparkline in the terminal."""
    data = mood_drift(buckets)
    values = [d["valence"] for d in data if d["valence"] is not None]
    if not values:
        console.print("[dim]No mood data yet.[/dim]")
        return

    # Normalise to sparkline chars
    chars = " ▁▂▃▄▅▆▇█"
    lo, hi = min(values), max(values)
    span = hi - lo or 1.0
    spark = "".join(chars[int((v - lo) / span * (len(chars) - 1))] for v in values)
    console.print(f"\n[bold]Emotional arc:[/bold] {spark}\n")
    console.print(f"[dim]neg {lo:.2f}  ←  →  pos {hi:.2f} · {len(values)} buckets[/dim]\n")
```

## What Changed: v3 → v4

### Architecture
- **`core.py` extracted** — no logic in CLI or UI; zero duplication between interfaces.
- **Lazy model loading** — Whisper and CLIP only load when the operation actually needs them. Text-only adds have no penalty.

### Data Integrity
- **`IndexIDMap2`** replaces `IndexFlatL2` — vectors are removed individually via `remove_ids()`. No more full rebuild on every delete.
- **Soft delete** — `deleted_at` timestamp; memories are recoverable. Hard delete available with `--hard`.
- **Embeddings stay in SQLite** for `rebuild_index` resilience; FAISS is treated as a hot cache, not source of truth.

### Add Flow
- **Media import is default** — `--no-import` to opt out. The vault is always self-contained unless you explicitly say otherwise.
- **Duplicate detection** — cosine similarity check against all active vectors before saving. Warns and prompts for confirmation.
- **Entity enrichment** — spaCy NER auto-appends people, places, orgs, events to tags.

### Search
- **Multimodal query** — `--image` or `--audio` on the CLI; file uploaders in the UI. Query embedding is fused exactly like storage embedding.
- **Resonance explanation** — each result includes a human-readable breakdown of its score.

### Export & Backup
