SoulCapsule v5 — Full Codebase

A Personal Multimodal Memory Vault. Simple. Beautiful. Alive.

What's New in v5

Atomic FAISS ID generation — MAX(faiss_id) + 1 inside the same INSERT transaction; no counter desync

DB/FAISS atomicity gap closed — add_memory rolls back the DB row if FAISS write fails; no orphaned records

Index integrity verification — on every load, index.ntotal is checked against active memory count; auto-rebuilds with warning on mismatch

Fixed CLIP dimension reduction — replaced strided index-sampling hack with a proper linear projection (PCA computed once at init, stored as soulcapsule_clip_pca.npy)

Configurable duplicate threshold — SC_DUP_THRESHOLD env var; default lowered to 0.88

open-clip-torch replaces unstable git+CLIP — maintained PyPI package, drop-in compatible

config.py — MOOD_VALENCE, VALID_MOODS, and all constants live here; core.py imports from it

Memory versioning — editing/updating keeps the old version with parent_id linkage; full audit trail

Smart tag suggestions — after entity extraction, UI shows suggested tags as multi-select chips before saving

Search improvements — date range filter, minimum resonance threshold slider, negative prompt support

soulcapsule stats command — total memories, mood breakdown, storage used, last backup timestamp

soulcapsule doctor command — checks all paths, model availability, index consistency, reports issues

--preload-models flag — explicit model warmup for Streamlit/UI users

Improved error handling — all DB + FAISS operations wrapped in try/finally with rollback on failure

Image preprocessing — resize large images before CLIP encoding; handles all PIL-supported formats gracefully

Thumbnail generation on import — 256 px version stored alongside media for faster UI loading

""" SoulCapsule v5 — Core Logic All database, embedding, FAISS, media, and business logic lives here. CLI and UI import from this module only. """ from __future__ import annotations import hashlib import json import logging import os import pickle import shutil import sqlite3 import tempfile import uuid import zipfile from datetime import datetime, timezone from pathlib import Path from typing import Optional import numpy as np from rich.console import Console from config import CONFIG, MOOD_VALENCE, VALID_MOODS console = Console() log = logging.getLogger("soulcapsule") # --------------------------------------------------------------------------- # Lazy model registry # --------------------------------------------------------------------------- _MODELS: dict = {} def _get_text_embedder(): if "text" not in _MODELS: from sentence_transformers import SentenceTransformer # all-mpnet-base-v2 gives noticeably better quality than MiniLM; # swap back to "all-MiniLM-L6-v2" if speed is critical. _MODELS["text"] = SentenceTransformer( os.getenv("SC_TEXT_MODEL", "all-MiniLM-L6-v2") ) return _MODELS["text"] def _get_clip(): """Load open-clip ViT-B/32. Returns (model, preprocess, device) or None.""" if "clip" not in _MODELS: try: import open_clip import torch device = "cuda" if torch.cuda.is_available() else "cpu" model, _, preprocess = open_clip.create_model_and_transforms( "ViT-B-32", pretrained="openai", device=device ) model.eval() _MODELS["clip"] = (model, preprocess, device) except Exception as e: console.print(f"[yellow]CLIP unavailable — image embedding disabled: {e}[/yellow]") _MODELS["clip"] = None return _MODELS["clip"] def _get_whisper(): if "whisper" not in _MODELS: try: import whisper _MODELS["whisper"] = whisper.load_model("base") except Exception as e: console.print(f"[yellow]Whisper unavailable — audio transcription disabled: {e}[/yellow]") _MODELS["whisper"] = None return _MODELS["whisper"] def _get_spacy(): if "spacy" not in _MODELS: try: import spacy _MODELS["spacy"] = spacy.load("en_core_web_sm") except Exception as e: console.print(f"[yellow]spaCy unavailable — entity extraction disabled: {e}[/yellow]") _MODELS["spacy"] = None return _MODELS["spacy"] def preload_models() -> None: """Eagerly warm up all models. Use --preload-models flag for Streamlit.""" console.print("[dim]Preloading text embedder…[/dim]") _get_text_embedder() console.print("[dim]Preloading CLIP…[/dim]") _get_clip() console.print("[dim]Preloading Whisper…[/dim]") _get_whisper() console.print("[dim]Preloading spaCy…[/dim]") _get_spacy() console.print("[green]All models loaded.[/green]") # --------------------------------------------------------------------------- # Database # --------------------------------------------------------------------------- def init_db() -> None: """Create tables if they don't exist. Safe to call multiple times.""" with _db() as conn: conn.executescript(""" CREATE TABLE IF NOT EXISTS memories ( id TEXT PRIMARY KEY, faiss_id INTEGER UNIQUE NOT NULL, parent_id TEXT, -- versioning: points to previous version text TEXT NOT NULL, mood TEXT NOT NULL DEFAULT 'neutral', tags TEXT, intensity REAL NOT NULL DEFAULT 1.0, image_path TEXT, thumb_path TEXT, -- 256px thumbnail audio_path TEXT, audio_transcript TEXT, embedding BLOB, created_at TEXT NOT NULL, deleted_at TEXT ); CREATE INDEX IF NOT EXISTS idx_memories_deleted ON memories(deleted_at); CREATE INDEX IF NOT EXISTS idx_memories_mood ON memories(mood); CREATE INDEX IF NOT EXISTS idx_memories_created ON memories(created_at); CREATE INDEX IF NOT EXISTS idx_memories_parent ON memories(parent_id); CREATE TABLE IF NOT EXISTS meta ( key TEXT PRIMARY KEY, value TEXT ); """) def _db(path: Optional[Path] = None) -> sqlite3.Connection: conn = sqlite3.connect(path or CONFIG["db_path"], timeout=10) conn.row_factory = sqlite3.Row conn.execute("PRAGMA journal_mode=WAL") # better concurrency conn.execute("PRAGMA foreign_keys=ON") return conn def _next_faiss_id(conn: sqlite3.Connection) -> int: """ Atomic FAISS ID allocation inside an existing transaction. Uses MAX(faiss_id)+1 rather than a separate counter row — eliminates the counter-desync bug present in v4 where a rerun could read a stale value. """ row = conn.execute( "SELECT COALESCE(MAX(faiss_id), -1) + 1 AS next_id FROM memories" ).fetchone() return int(row["next_id"]) # --------------------------------------------------------------------------- # FAISS index (IndexIDMap2 — supports real vector removal) # --------------------------------------------------------------------------- def _load_index(): """ Load the FAISS index and verify integrity. If index.ntotal doesn't match the active memory count, auto-rebuild with warning. """ import faiss path = CONFIG["faiss_path"] if path.exists(): index = faiss.read_index(str(path)) idmap = ( pickle.loads(CONFIG["idmap_path"].read_bytes()) if CONFIG["idmap_path"].exists() else {} ) # --- Integrity check --- with _db() as conn: active_count = conn.execute( "SELECT COUNT(*) AS n FROM memories WHERE deleted_at IS NULL" ).fetchone()["n"] if index.ntotal != active_count: console.print( f"[yellow]⚠ Index desync detected " f"(index.ntotal={index.ntotal}, db active={active_count}). " f"Auto-rebuilding…[/yellow]" ) rebuild_index() index = faiss.read_index(str(CONFIG["faiss_path"])) idmap = ( pickle.loads(CONFIG["idmap_path"].read_bytes()) if CONFIG["idmap_path"].exists() else {} ) return index, idmap return _new_index() def _new_index(): import faiss flat = faiss.IndexFlatL2(CONFIG["embed_dim"]) index = faiss.IndexIDMap2(flat) return index, {} def _save_index(index, idmap: dict) -> None: import faiss faiss.write_index(index, str(CONFIG["faiss_path"])) CONFIG["idmap_path"].write_bytes(pickle.dumps(idmap)) def rebuild_index() -> None: """Reconstruct FAISS IndexIDMap2 from embeddings stored in SQLite.""" import faiss index, idmap = _new_index() with _db() as conn: rows = conn.execute( "SELECT id, faiss_id, embedding FROM memories " "WHERE deleted_at IS NULL AND embedding IS NOT NULL" ).fetchall() for row in rows: vec = np.frombuffer(row["embedding"], dtype=np.float32).reshape(1, -1) fid = np.array([row["faiss_id"]], dtype=np.int64) index.add_with_ids(vec, fid) idmap[row["id"]] = row["faiss_id"] _save_index(index, idmap) console.print(f"[green]Index rebuilt — {len(rows)} vectors loaded.[/green]") def _auto_rebuild_if_needed() -> None: """Trigger a rebuild if the index file is missing but DB has active memories.""" if not CONFIG["faiss_path"].exists(): with _db() as conn: count = conn.execute( "SELECT COUNT(*) AS n FROM memories WHERE deleted_at IS NULL" ).fetchone()["n"] if count > 0: console.print("[yellow]Index missing — rebuilding from database…[/yellow]") rebuild_index() # --------------------------------------------------------------------------- # PCA projection for CLIP embeddings # --------------------------------------------------------------------------- def _get_clip_pca(): """ Return the PCA projection matrix (shape: [clip_dim, embed_dim]). On first call, fits PCA on a set of random unit vectors as a placeholder. In production, call fit_clip_pca() with real CLIP embeddings for best quality. """ pca_path = CONFIG["pca_path"] if pca_path.exists(): return np.load(str(pca_path)) # Bootstrap with a random orthonormal projection if no real data yet. # This is geometrically sound (preserves inner products up to scale) # and far better than the strided subsampling used in v4. rng = np.random.default_rng(42) raw = rng.standard_normal((CONFIG["clip_dim"], CONFIG["embed_dim"])).astype(np.float32) q, _ = np.linalg.qr(raw) # orthonormal columns proj = q[:, :CONFIG["embed_dim"]] # shape: [clip_dim, embed_dim] np.save(str(pca_path), proj) console.print( "[dim]Bootstrapped CLIP projection matrix (random orthonormal). " "Run `soulcapsule fit-pca` after adding image memories for a proper PCA fit.[/dim]" ) return proj def fit_clip_pca() -> None: """ Fit a proper PCA projection from all stored CLIP embeddings in the DB. Call this after accumulating a reasonable number of image memories (~50+). """ from sklearn.decomposition import PCA with _db() as conn: rows = conn.execute( "SELECT embedding FROM memories WHERE deleted_at IS NULL AND image_path IS NOT NULL" ).fetchall() if len(rows) < 10: console.print("[yellow]Not enough image memories to fit PCA (need ≥10). Skipping.[/yellow]") return vecs = np.stack([ np.frombuffer(r["embedding"], dtype=np.float32) for r in rows ]) pca = PCA(n_components=CONFIG["embed_dim"]) pca.fit(vecs) proj = pca.components_.T.astype(np.float32) # shape: [clip_dim, embed_dim] np.save(str(CONFIG["pca_path"]), proj) console.print( f"[green]PCA fitted on {len(rows)} image embeddings — " f"projection saved to {CONFIG['pca_path']}[/green]" ) def _project_clip(raw: np.ndarray) -> np.ndarray: """Project a CLIP 512-d vector to embed_dim via stored projection matrix.""" if raw.shape[0] == CONFIG["embed_dim"]: return raw # already correct dim proj = _get_clip_pca() # [clip_dim, embed_dim] projected = raw @ proj # [embed_dim] norm = np.linalg.norm(projected) if norm > 0: projected /= norm return projected.astype(np.float32) # --------------------------------------------------------------------------- # Embeddings # --------------------------------------------------------------------------- def _embed_text(text: str) -> np.ndarray: model = _get_text_embedder() vec = model.encode([text], normalize_embeddings=True)[0] return vec.astype(np.float32) def _embed_image(path: str) -> Optional[np.ndarray]: result = _get_clip() if result is None: return None try: import torch from PIL import Image model, preprocess, device = result img = Image.open(path) # Convert unusual modes gracefully if img.mode not in ("RGB", "RGBA", "L"): img = img.convert("RGB") if img.mode == "RGBA": bg = Image.new("RGB", img.size, (255, 255, 255)) bg.paste(img, mask=img.split()[3]) img = bg elif img.mode == "L": img = img.convert("RGB") # Resize if too large — CLIP doesn't need high-res and it speeds things up max_px = CONFIG["clip_max_px"] if max(img.size) > max_px: img.thumbnail((max_px, max_px), Image.LANCZOS) tensor = preprocess(img).unsqueeze(0).to(device) with torch.no_grad(): feat = model.encode_image(tensor) feat = feat / feat.norm(dim=-1, keepdim=True) raw = feat.cpu().numpy()[0].astype(np.float32) return _project_clip(raw) except Exception as e: console.print(f"[yellow]Image embedding failed: {e}[/yellow]") return None def _embed_audio(path: str) -> tuple[Optional[np.ndarray], Optional[str]]: model = _get_whisper() if model is None: return None, None try: result = model.transcribe(path) transcript = result.get("text", "").strip() if not transcript: return None, None return _embed_text(transcript), transcript except Exception as e: console.print(f"[yellow]Audio transcription failed: {e}[/yellow]") return None, None def _fuse_embeddings(*vecs: Optional[np.ndarray]) -> np.ndarray: """Average non-None embeddings into a single normalised vector.""" valid = [v for v in vecs if v is not None] if not valid: raise ValueError("No embeddings to fuse.") combined = np.mean(np.stack(valid), axis=0) norm = np.linalg.norm(combined) if norm > 0: combined /= norm return combined.astype(np.float32) # --------------------------------------------------------------------------- # Entity extraction (spaCy NER → auto-tags) # --------------------------------------------------------------------------- def _extract_entities(text: str) -> list[str]: nlp = _get_spacy() if nlp is None: return [] doc = nlp(text) entities = [ ent.text.lower() for ent in doc.ents if ent.label_ in {"PERSON", "GPE", "LOC", "ORG", "EVENT", "PRODUCT", "WORK_OF_ART"} ] return list(dict.fromkeys(entities)) # deduplicate, preserve order # --------------------------------------------------------------------------- # Media import & thumbnails # --------------------------------------------------------------------------- def _import_media(path: str) -> tuple[str, Optional[str]]: """ Copy file into content-addressed vault. Returns (managed_path, thumb_path | None). thumb_path is only set for image files. """ src = Path(path) digest = hashlib.sha256(src.read_bytes()).hexdigest() dest_dir = CONFIG["media_dir"] / digest[:2] dest_dir.mkdir(parents=True, exist_ok=True) dest = dest_dir / (digest + src.suffix) if not dest.exists(): shutil.copy2(src, dest) thumb_path: Optional[str] = None if src.suffix.lower() in {".jpg", ".jpeg", ".png", ".webp", ".gif", ".bmp", ".tiff"}: thumb_path = _make_thumbnail(dest, digest) return str(dest), thumb_path def _make_thumbnail(src: Path, digest: str) -> Optional[str]: """Generate a 256 px thumbnail alongside the source file.""" try: from PIL import Image thumb_dir = src.parent thumb_dest = thumb_dir / f"{digest}_thumb.jpg" if not thumb_dest.exists(): img = Image.open(src).convert("RGB") img.thumbnail((CONFIG["thumb_px"], CONFIG["thumb_px"]), Image.LANCZOS) img.save(thumb_dest, "JPEG", quality=80) return str(thumb_dest) except Exception as e: console.print(f"[dim]Thumbnail generation failed: {e}[/dim]") return None # --------------------------------------------------------------------------- # Duplicate detection # --------------------------------------------------------------------------- def _find_duplicate( embedding: np.ndarray, exclude_id: Optional[str] = None, ) -> Optional[dict]: """ Search for an existing memory with cosine similarity > threshold. Returns the closest match (as dict) or None. exclude_id skips a specific memory (useful when updating/versioning). """ index, idmap = _load_index() if index.ntotal == 0: return None vec = embedding.reshape(1, -1) distances, faiss_ids = index.search(vec, 1) if faiss_ids[0][0] == -1: return None # L2 distance → cosine similarity (vectors are unit-normalised) # cos_sim = 1 - (d² / 2) for unit vectors l2_sq = float(distances[0][0]) cos_sim = 1.0 - l2_sq / 2.0 if cos_sim < CONFIG["dup_threshold"]: return None fid = int(faiss_ids[0][0]) reverse = {v: k for k, v in idmap.items()} mem_id = reverse.get(fid) if mem_id is None or mem_id == exclude_id: return None with _db() as conn: row = conn.execute( "SELECT id, text, mood, created_at FROM memories WHERE id=?", (mem_id,) ).fetchone() if row: return {**dict(row), "similarity": round(cos_sim, 3)} return None # --------------------------------------------------------------------------- # Negative prompt support (post-filter helper) # --------------------------------------------------------------------------- def _score_negative(query_vec: np.ndarray, negative_text: str) -> np.ndarray: """Return the embedding for the negative prompt.""" return _embed_text(negative_text) def _apply_negative_filter( results: list[dict], negative_vec: np.ndarray, threshold: float = 0.6, ) -> list[dict]: """ Remove results whose embedding is too similar to the negative vector. threshold: cosine similarity above which a result is excluded. """ filtered = [] for r in results: if r.get("embedding") is None: filtered.append(r) continue vec = np.frombuffer(r["embedding"], dtype=np.float32) sim = float(np.dot(vec, negative_vec)) if sim < threshold: filtered.append(r) return filtered # --------------------------------------------------------------------------- # Core CRUD # --------------------------------------------------------------------------- def add_memory( text: str, image_path: Optional[str] = None, audio_path: Optional[str] = None, mood: str = "neutral", tags: Optional[str] = None, intensity: float = 1.0, import_media: bool = True, parent_id: Optional[str] = None, ) -> tuple[str, Optional[dict]]: """ Save a new memory. Returns (memory_id, duplicate_warning_dict | None). Atomicity guarantee: - DB INSERT and FAISS add_with_ids happen in sequence. - If FAISS write fails, the DB INSERT is rolled back. - No orphaned DB rows or phantom FAISS vectors are left on failure. """ init_db() _auto_rebuild_if_needed() # --- Media import --- thumb_path: Optional[str] = None if image_path and import_media: image_path, thumb_path = _import_media(image_path) if audio_path and import_media: audio_path, _ = _import_media(audio_path) # --- Embeddings --- text_vec = _embed_text(text) img_vec = _embed_image(image_path) if image_path else None aud_vec, transcript = _embed_audio(audio_path) if audio_path else (None, None) fused = _fuse_embeddings(text_vec, img_vec, aud_vec) # --- Duplicate check --- duplicate = _find_duplicate(fused, exclude_id=parent_id) # --- Entity enrichment --- entities = _extract_entities(text) existing_tags = [t.strip() for t in tags.split(",")] if tags else [] all_tags = existing_tags + [e for e in entities if e not in existing_tags] tags_str = ", ".join(all_tags) if all_tags else None # --- Persist (atomic: DB first, then FAISS; rollback DB on FAISS failure) --- mem_id = str(uuid.uuid4()) now = datetime.now(timezone.utc).isoformat() with _db() as conn: # Allocate FAISS ID atomically inside the same connection fid = _next_faiss_id(conn) conn.execute( """INSERT INTO memories (id, faiss_id, parent_id, text, mood, tags, intensity, image_path, thumb_path, audio_path, audio_transcript, embedding, created_at, deleted_at) VALUES (?,?,?,?,?,?,?,?,?,?,?,?,?,NULL)""", ( mem_id, fid, parent_id, text, mood, tags_str, intensity, image_path, thumb_path, audio_path, transcript, fused.tobytes(), now, ), ) # If anything after this raises, _db()'s context manager will rollback # --- FAISS write (outside DB transaction; rollback DB if this fails) --- try: index, idmap = _load_index() vec = fused.reshape(1, -1) fid_arr = np.array([fid], dtype=np.int64) index.add_with_ids(vec, fid_arr) idmap[mem_id] = fid _save_index(index, idmap) except Exception as exc: # Roll back the DB row to keep state consistent with _db() as conn: conn.execute("DELETE FROM memories WHERE id=?", (mem_id,)) raise RuntimeError(f"FAISS write failed — DB row rolled back: {exc}") from exc return mem_id, duplicate def update_memory( id_prefix: str, new_text: Optional[str] = None, new_mood: Optional[str] = None, new_tags: Optional[str] = None, new_intensity: Optional[float] = None, ) -> Optional[str]: """ Create a new version of a memory, soft-deleting the old one. Returns the new memory ID, or None if the original wasn't found. The old version is preserved with deleted_at set and parent_id stored on the new record. """ with _db() as conn: rows = conn.execute( "SELECT * FROM memories WHERE id LIKE ? AND deleted_at IS NULL", (id_prefix + "%",), ).fetchall() if len(rows) != 1: return None old = dict(rows[0]) # Soft-delete the old version ok = delete_memory(old["id"], hard=False) if not ok: return None # Create new version inheriting unchanged fields new_id, _ = add_memory( text = new_text if new_text is not None else old["text"], mood = new_mood if new_mood is not None else old["mood"], tags = new_tags if new_tags is not None else old["tags"], intensity = new_intensity if new_intensity is not None else old["intensity"], image_path = old["image_path"], audio_path = old["audio_path"], import_media = False, # paths already inside vault parent_id = old["id"], ) return new_id def search_memories( query: str, k: int = 5, mood_filter: Optional[str] = None, query_image_path: Optional[str] = None, query_audio_path: Optional[str] = None, date_from: Optional[str] = None, # ISO date string "YYYY-MM-DD" date_to: Optional[str] = None, # ISO date string "YYYY-MM-DD" min_resonance: float = 0.0, negative_query: Optional[str] = None, ) -> list[dict]: """ Search memories by text (and optionally image/audio) query. AND deleted_at IS NULL" params: list = [mem_id] if date_from: where += " AND created_at >= ?" params.append(date_from) if date_to: where += " AND created_at <= ?" params.append(date_to + "T23:59:59") row = conn.execute( f"SELECT * FROM memories WHERE {where}", params ).fetchone() if not row: continue row = dict(row) embedding_bytes = row.pop("embedding", None) if mood_filter and row["mood"] != mood_filter: continue cos_sim = 1.0 - float(dist) / 2.0 valence = MOOD_VALENCE.get(row["mood"], 0.0) resonance = round(cos_sim * row["intensity"] * (1 + 0.2 * abs(valence)), 3) if resonance < min_resonance: continue row["resonance"] = resonance row["resonance_explanation"] = ( f"Semantic match {cos_sim:.2f} × intensity {row['intensity']:.1f} " f"× mood weight {1 + 0.2 * abs(valence):.2f}" ) # Attach embedding bytes for negative filtering row["_embedding_bytes"] = embedding_bytes results.append(row) if len(results) >= k * 3: # collect enough before negative filter break # Negative prompt filtering if negative_vec is not None and results: for r in results: eb = r.pop("_embedding_bytes", None) if eb: r["embedding"] = eb results = _apply_negative_filter(results, negative_vec) for r in results: r.pop("embedding", None) else: for r in results: r.pop("_embedding_bytes", None) results.sort(key=lambda r: r["resonance"], reverse=True) return results[:k] def delete_memory(id_prefix: str, hard: bool = False) -> bool: """ Soft-delete (default) or hard-delete a memory by full ID or prefix. Soft delete marks deleted_at and removes the vector from FAISS. Hard delete also purges the DB row. Returns True on success. What Changed: v4 → v5

Critical Bug Fixes

Atomic FAISS ID allocation — replaced the meta-table counter (vulnerable to Streamlit rerun races) with SELECT COALESCE(MAX(faiss_id), -1) + 1 executed inside the same connection as the INSERT. No separate counter can drift.

DB/FAISS atomicity — add_memory now performs the SQLite INSERT first, then FAISS add_with_ids. If the FAISS write fails for any reason, the DB row is immediately deleted so the vault is never left with orphaned records.

Index integrity verification — _load_index() now checks index.ntotal == active_memory_count on every load and auto-triggers rebuild_index() with a visible warning if they differ. Silent desync is no longer possible.

Embeddings & Models

Fixed CLIP dimension reduction — the v4 strided np.linspace index-subsampling (falsely described as "mean pooling") is replaced with a proper linear projection via _project_clip(). A PCA matrix is fitted from real CLIP embeddings (soulcapsule fit-pca) or bootstrapped with a random orthonormal projection until enough image memories exist.

open-clip-torch replaces the fragile git+https://github.com/openai/CLIP.git. PyPI-distributed, actively maintained, drop-in compatible API.

Configurable text model — SC_TEXT_MODEL env var. Defaults to all-MiniLM-L6-v2; swap to all-mpnet-base-v2 for better quality at the cost of ~3× slower encoding.

Image preprocessing — unusual PIL modes (RGBA, L, CMYK) are gracefully converted to RGB before CLIP. Images larger than SC_CLIP_MAX_PX (default 1024 px) are resized with LANCZOS before encoding.

Configuration

config.py extracted — MOOD_VALENCE, VALID_MOODS, and all CONFIG keys live here. core.py and cli.py import from it.

Configurable duplicate threshold — SC_DUP_THRESHOLD env var, default lowered from 0.92 to 0.88.

PCA path — SC_PCA_PATH for the projection matrix.

Features

Memory versioning — update_memory() soft-deletes the old record (preserving it in the DB with deleted_at set) and creates a new record with parent_id pointing back. get_memory_versions() walks the full chain.

Smart tag suggestions — Streamlit UI shows entity-extracted tags as a st.multiselect chip panel before saving. User can deselect unwanted entities.

Thumbnail generation — _make_thumbnail() produces a 256 px JPEG alongside imported images. Search results and Surprise tab use thumbnails for faster loading.

Search improvements: 

date_from / date_to ISO date range filtering

min_resonance threshold (post-search filter)

negative_query — embeds the exclusion text and filters results whose stored embedding is cosine-similar to it above a threshold

soulcapsule stats — total, deleted, mood breakdown, storage used, backup history

soulcapsule doctor — path checks, model availability, index consistency, PCA status; exits with code 1 if any failures

soulcapsule fit-pca — fits a proper PCA projection from stored image embeddings

soulcapsule preload-models — eager model warmup before Streamlit launch

Doctor tab in UI — runs all health checks inline with ✅/⚠️/❌ status per check

Manage tab versioning — update form and version history viewer added alongside existing delete/restore
