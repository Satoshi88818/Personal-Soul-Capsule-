SoulCapsule v6 — Full Codebase

A Personal Multimodal Memory Vault. Simple. Beautiful. Alive.

What's New in v6

Media garbage collection — soulcapsule gc command purges orphaned media files after hard deletes; hard delete now accepts --purge-media flag

Soft negative prompt scoring — negative query is now a continuous penalty in resonance scoring rather than a hard binary filter; smoother, more nuanced exclusions

Top-3 duplicate detection — _find_duplicate() checks the nearest 3 candidates and returns the best match above threshold (was nearest-1)

User-configurable mood weighting — SC_MOOD_WEIGHT_SIGNED env var switches valence boost from abs(valence) to signed valence, so positive memories are boosted and negative ones are penalised (or keep the old symmetric behaviour)

Recency boost — SC_RECENCY_BOOST and --recency-boost search option; optional exponential decay that rewards recently-created memories

Static HTML export — soulcapsule export --format html generates a self-contained browsable HTML archive (no server needed; works on mobile)

soulcapsule gc — standalone garbage-collect command for orphaned vault media

Hierarchical / prefixed tags — tags now support category:value syntax; search and UI handle the colon separator gracefully

All v5 features retained and fully compatible

""" SoulCapsule v6 — Core Logic All database, embedding, FAISS, media, GC, and business logic lives here. CLI and UI import from this module only. """ from __future__ import annotations import hashlib import json import logging import os import pickle import shutil import sqlite3 import tempfile import uuid import zipfile from datetime import datetime, timezone, timedelta from pathlib import Path from typing import Optional import numpy as np from rich.console import Console from config import CONFIG, MOOD_VALENCE, VALID_MOODS console = Console() log = logging.getLogger("soulcapsule") # --------------------------------------------------------------------------- # Lazy model registry # --------------------------------------------------------------------------- _MODELS: dict = {} def _get_text_embedder(): if "text" not in _MODELS: from sentence_transformers import SentenceTransformer _MODELS["text"] = SentenceTransformer( os.getenv("SC_TEXT_MODEL", "all-MiniLM-L6-v2") ) return _MODELS["text"] def _get_clip(): """Load open-clip ViT-B/32. Returns (model, preprocess, device) or None.""" if "clip" not in _MODELS: try: import open_clip import torch device = "cuda" if torch.cuda.is_available() else "cpu" model, _, preprocess = open_clip.create_model_and_transforms( "ViT-B-32", pretrained="openai", device=device ) model.eval() _MODELS["clip"] = (model, preprocess, device) except Exception as e: console.print(f"[yellow]CLIP unavailable — image embedding disabled: {e}[/yellow]") _MODELS["clip"] = None return _MODELS["clip"] def _get_whisper(): if "whisper" not in _MODELS: try: import whisper _MODELS["whisper"] = whisper.load_model("base") except Exception as e: console.print(f"[yellow]Whisper unavailable — audio transcription disabled: {e}[/yellow]") _MODELS["whisper"] = None return _MODELS["whisper"] def _get_spacy(): if "spacy" not in _MODELS: try: import spacy _MODELS["spacy"] = spacy.load("en_core_web_sm") except Exception as e: console.print(f"[yellow]spaCy unavailable — entity extraction disabled: {e}[/yellow]") _MODELS["spacy"] = None return _MODELS["spacy"] def preload_models() -> None: """Eagerly warm up all models. Use --preload-models flag for Streamlit.""" console.print("[dim]Preloading text embedder…[/dim]") _get_text_embedder() console.print("[dim]Preloading CLIP…[/dim]") _get_clip() console.print("[dim]Preloading Whisper…[/dim]") _get_whisper() console.print("[dim]Preloading spaCy…[/dim]") _get_spacy() console.print("[green]All models loaded.[/green]") # --------------------------------------------------------------------------- # Database # --------------------------------------------------------------------------- def init_db() -> None: """Create tables if they don't exist. Safe to call multiple times.""" with _db() as conn: conn.executescript(""" CREATE TABLE IF NOT EXISTS memories ( id TEXT PRIMARY KEY, faiss_id INTEGER UNIQUE NOT NULL, parent_id TEXT, text TEXT NOT NULL, mood TEXT NOT NULL DEFAULT 'neutral', tags TEXT, intensity REAL NOT NULL DEFAULT 1.0, image_path TEXT, thumb_path TEXT, audio_path TEXT, audio_transcript TEXT, embedding BLOB, created_at TEXT NOT NULL, deleted_at TEXT ); CREATE INDEX IF NOT EXISTS idx_memories_deleted ON memories(deleted_at); CREATE INDEX IF NOT EXISTS idx_memories_mood ON memories(mood); CREATE INDEX IF NOT EXISTS idx_memories_created ON memories(created_at); CREATE INDEX IF NOT EXISTS idx_memories_parent ON memories(parent_id); CREATE TABLE IF NOT EXISTS meta ( key TEXT PRIMARY KEY, value TEXT ); """) def _db(path: Optional[Path] = None) -> sqlite3.Connection: conn = sqlite3.connect(path or CONFIG["db_path"], timeout=10) conn.row_factory = sqlite3.Row conn.execute("PRAGMA journal_mode=WAL") conn.execute("PRAGMA foreign_keys=ON") return conn def _next_faiss_id(conn: sqlite3.Connection) -> int: """ Atomic FAISS ID allocation inside an existing transaction. Uses MAX(faiss_id)+1 rather than a separate counter row. """ row = conn.execute( "SELECT COALESCE(MAX(faiss_id), -1) + 1 AS next_id FROM memories" ).fetchone() return int(row["next_id"]) # --------------------------------------------------------------------------- # FAISS index (IndexIDMap2 — supports real vector removal) # --------------------------------------------------------------------------- def _load_index(): """ Load the FAISS index and verify integrity. If index.ntotal doesn't match the active memory count, auto-rebuild. """ import faiss path = CONFIG["faiss_path"] if path.exists(): index = faiss.read_index(str(path)) idmap = ( pickle.loads(CONFIG["idmap_path"].read_bytes()) if CONFIG["idmap_path"].exists() else {} ) with _db() as conn: active_count = conn.execute( "SELECT COUNT(*) AS n FROM memories WHERE deleted_at IS NULL" ).fetchone()["n"] if index.ntotal != active_count: console.print( f"[yellow]⚠ Index desync detected " f"(index.ntotal={index.ntotal}, db active={active_count}). " f"Auto-rebuilding…[/yellow]" ) rebuild_index() index = faiss.read_index(str(CONFIG["faiss_path"])) idmap = ( pickle.loads(CONFIG["idmap_path"].read_bytes()) if CONFIG["idmap_path"].exists() else {} ) return index, idmap return _new_index() def _new_index(): import faiss flat = faiss.IndexFlatL2(CONFIG["embed_dim"]) index = faiss.IndexIDMap2(flat) return index, {} def _save_index(index, idmap: dict) -> None: import faiss faiss.write_index(index, str(CONFIG["faiss_path"])) CONFIG["idmap_path"].write_bytes(pickle.dumps(idmap)) def rebuild_index() -> None: """Reconstruct FAISS IndexIDMap2 from embeddings stored in SQLite.""" import faiss index, idmap = _new_index() with _db() as conn: rows = conn.execute( "SELECT id, faiss_id, embedding FROM memories " "WHERE deleted_at IS NULL AND embedding IS NOT NULL" ).fetchall() for row in rows: vec = np.frombuffer(row["embedding"], dtype=np.float32).reshape(1, -1) fid = np.array([row["faiss_id"]], dtype=np.int64) index.add_with_ids(vec, fid) idmap[row["id"]] = row["faiss_id"] _save_index(index, idmap) console.print(f"[green]Index rebuilt — {len(rows)} vectors loaded.[/green]") def _auto_rebuild_if_needed() -> None: if not CONFIG["faiss_path"].exists(): with _db() as conn: count = conn.execute( "SELECT COUNT(*) AS n FROM memories WHERE deleted_at IS NULL" ).fetchone()["n"] if count > 0: console.print("[yellow]Index missing — rebuilding from database…[/yellow]") rebuild_index() # --------------------------------------------------------------------------- # PCA projection for CLIP embeddings # --------------------------------------------------------------------------- def _get_clip_pca(): """ Return the PCA projection matrix (shape: [clip_dim, embed_dim]). Bootstraps with a random orthonormal projection if no real data exists yet. """ pca_path = CONFIG["pca_path"] if pca_path.exists(): return np.load(str(pca_path)) rng = np.random.default_rng(42) raw = rng.standard_normal((CONFIG["clip_dim"], CONFIG["embed_dim"])).astype(np.float32) q, _ = np.linalg.qr(raw) proj = q[:, :CONFIG["embed_dim"]] np.save(str(pca_path), proj) console.print( "[dim]Bootstrapped CLIP projection matrix (random orthonormal). " "Run `soulcapsule fit-pca` after adding image memories for a proper PCA fit.[/dim]" ) return proj def fit_clip_pca() -> None: """ Fit a proper PCA projection from all stored CLIP embeddings in the DB. Recommended after accumulating ~50+ image memories. """ from sklearn.decomposition import PCA with _db() as conn: rows = conn.execute( "SELECT embedding FROM memories WHERE deleted_at IS NULL AND image_path IS NOT NULL" ).fetchall() if len(rows) < 10: console.print("[yellow]Not enough image memories to fit PCA (need ≥10). Skipping.[/yellow]") return vecs = np.stack([np.frombuffer(r["embedding"], dtype=np.float32) for r in rows]) pca = PCA(n_components=CONFIG["embed_dim"]) pca.fit(vecs) proj = pca.components_.T.astype(np.float32) np.save(str(CONFIG["pca_path"]), proj) console.print( f"[green]PCA fitted on {len(rows)} image embeddings — " f"projection saved to {CONFIG['pca_path']}[/green]" ) def _project_clip(raw: np.ndarray) -> np.ndarray: if raw.shape[0] == CONFIG["embed_dim"]: return raw proj = _get_clip_pca() projected = raw @ proj norm = np.linalg.norm(projected) if norm > 0: projected /= norm return projected.astype(np.float32) # --------------------------------------------------------------------------- # Embeddings # --------------------------------------------------------------------------- def _embed_text(text: str) -> np.ndarray: model = _get_text_embedder() vec = model.encode([text], normalize_embeddings=True)[0] return vec.astype(np.float32) def _embed_image(path: str) -> Optional[np.ndarray]: result = _get_clip() if result is None: return None try: import torch from PIL import Image model, preprocess, device = result img = Image.open(path) if img.mode == "RGBA": bg = Image.new("RGB", img.size, (255, 255, 255)) bg.paste(img, mask=img.split()[3]) img = bg elif img.mode != "RGB": img = img.convert("RGB") max_px = CONFIG["clip_max_px"] if max(img.size) > max_px: img.thumbnail((max_px, max_px), Image.LANCZOS) tensor = preprocess(img).unsqueeze(0).to(device) with torch.no_grad(): feat = model.encode_image(tensor) feat = feat / feat.norm(dim=-1, keepdim=True) raw = feat.cpu().numpy()[0].astype(np.float32) return _project_clip(raw) except Exception as e: console.print(f"[yellow]Image embedding failed: {e}[/yellow]") return None def _embed_audio(path: str) -> tuple[Optional[np.ndarray], Optional[str]]: model = _get_whisper() if model is None: return None, None try: result = model.transcribe(path) transcript = result.get("text", "").strip() if not transcript: return None, None return _embed_text(transcript), transcript except Exception as e: console.print(f"[yellow]Audio transcription failed: {e}[/yellow]") return None, None def _fuse_embeddings(*vecs: Optional[np.ndarray]) -> np.ndarray: """Average non-None embeddings into a single normalised vector.""" valid = [v for v in vecs if v is not None] if not valid: raise ValueError("No embeddings to fuse.") combined = np.mean(np.stack(valid), axis=0) norm = np.linalg.norm(combined) if norm > 0: combined /= norm return combined.astype(np.float32) # --------------------------------------------------------------------------- # Entity extraction (spaCy NER → auto-tags) # --------------------------------------------------------------------------- def _extract_entities(text: str) -> list[str]: nlp = _get_spacy() if nlp is None: return [] doc = nlp(text) entities = [ ent.text.lower() for ent in doc.ents if ent.label_ in {"PERSON", "GPE", "LOC", "ORG", "EVENT", "PRODUCT", "WORK_OF_ART"} ] return list(dict.fromkeys(entities)) # --------------------------------------------------------------------------- # Media import & thumbnails # --------------------------------------------------------------------------- def _import_media(path: str) -> tuple[str, Optional[str]]: """ Copy file into content-addressed vault. Returns (managed_path, thumb_path | None). """ src = Path(path) digest = hashlib.sha256(src.read_bytes()).hexdigest() dest_dir = CONFIG["media_dir"] / digest[:2] dest_dir.mkdir(parents=True, exist_ok=True) dest = dest_dir / (digest + src.suffix) if not dest.exists(): shutil.copy2(src, dest) thumb_path: Optional[str] = None if src.suffix.lower() in {".jpg", ".jpeg", ".png", ".webp", ".gif", ".bmp", ".tiff"}: thumb_path = _make_thumbnail(dest, digest) return str(dest), thumb_path def _make_thumbnail(src: Path, digest: str) -> Optional[str]: """Generate a 256 px thumbnail alongside the source file.""" try: from PIL import Image thumb_dest = src.parent / f"{digest}_thumb.jpg" if not thumb_dest.exists(): img = Image.open(src).convert("RGB") img.thumbnail((CONFIG["thumb_px"], CONFIG["thumb_px"]), Image.LANCZOS) img.save(thumb_dest, "JPEG", quality=80) return str(thumb_dest) except Exception as e: console.print(f"[dim]Thumbnail generation failed: {e}[/dim]") return None # --------------------------------------------------------------------------- # v6: Media garbage collection # --------------------------------------------------------------------------- def collect_garbage(dry_run: bool = False) -> dict: """ Identify and optionally remove media files in the vault that are no longer referenced by any memory (active or soft-deleted). Checks the nearest SC_DUP_TOP_K (default 3) candidates and returns the single best match above threshold, or None. exclude_id skips a specific memory (useful when updating/versioning). """ index, idmap = _load_index() if index.ntotal == 0: return None top_k = CONFIG["dup_top_k"] vec = embedding.reshape(1, -1) distances, faiss_ids = index.search(vec, min(top_k, index.ntotal)) reverse = {v: k for k, v in idmap.items()} best: Optional[dict] = None best_sim = -1.0 for dist, fid in zip(distances[0], faiss_ids[0]): if fid == -1: continue l2_sq = float(dist) cos_sim = 1.0 - l2_sq / 2.0 if cos_sim < CONFIG["dup_threshold"]: continue mem_id = reverse.get(int(fid)) if mem_id is None or mem_id == exclude_id: continue if cos_sim > best_sim: with _db() as conn: row = conn.execute( "SELECT id, text, mood, created_at FROM memories WHERE id=?", (mem_id,), ).fetchone() if row: best = {**dict(row), "similarity": round(cos_sim, 3)} best_sim = cos_sim return best # --------------------------------------------------------------------------- # v6: Resonance scoring helpers # --------------------------------------------------------------------------- def _mood_weight(valence: float) -> float: """ Compute the mood multiplier for resonance scoring. mode="abs" → 1 + 0.2 * |valence| (v5 behaviour; symmetric) mode="signed" → 1 + 0.2 * valence (v6 opt-in; boosts positive, penalises negative) """ mode = CONFIG.get("mood_weight_mode", "abs") if mode == "signed": return 1.0 + 0.2 * valence return 1.0 + 0.2 * abs(valence) def _recency_multiplier(created_at_iso: str) -> float: """ Optional exponential recency boost. """ halflife = CONFIG.get("recency_boost_halflife", 0.0) if halflife <= 0: return 1.0 try: created = datetime.fromisoformat(created_at_iso) age_days = (datetime.now(timezone.utc) - created).total_seconds() / 86400 return float(np.exp(-np.log(2) / halflife * age_days)) except Exception: return 1.0 def _negative_penalty(result_vec: np.ndarray, negative_vec: np.ndarray) -> float: """ Return a soft penalty scalar in [0, 1] representing how similar result_vec is to the negative embedding. penalty = weight * max(0, cos_sim) This is subtracted from resonance rather than acting as a hard filter. """ weight = CONFIG.get("negative_penalty_weight", 0.5) if weight <= 0: return 0.0 cos_sim = float(np.dot(result_vec, negative_vec)) return weight * max(0.0, cos_sim) # --------------------------------------------------------------------------- # Core CRUD # --------------------------------------------------------------------------- def add_memory( text: str, image_path: Optional[str] = None, audio_path: Optional[str] = None, mood: str = "neutral", tags: Optional[str] = None, intensity: float = 1.0, import_media: bool = True, parent_id: Optional[str] = None, ) -> tuple[str, Optional[dict]]: """ Save a new memory. What Changed: v5 → v6

New Features

Media Garbage Collection (collect_garbage(), soulcapsule gc) Hard-deleting memories without --purge-media previously left media files stranded in the vault indefinitely. v6 adds collect_garbage() in core.py which walks all files in soulcapsule_media/, compares them against every path referenced in the DB (including soft-deleted rows), and removes unlinked files. The CLI exposes this as soulcapsule gc with a --dry-run flag. delete_memory() now also accepts --purge-media for immediate cleanup on hard delete.

Soft Negative Prompt Scoring The v5 negative query was a binary filter: results with cosine similarity above a threshold to the exclusion embedding were simply dropped. v6 changes this to a continuous penalty: resonance -= weight * max(0, cos_sim_to_negative). This means memories strongly related to the exclusion term score lower, but memories only loosely related are not hard-cut. The penalty weight is configurable via SC_NEG_PENALTY (default 0.5).

Top-3 Duplicate Detection _find_duplicate() now fetches the nearest SC_DUP_TOP_K (default 3) candidates from FAISS and returns the highest-similarity match above threshold, rather than only checking the single nearest neighbour. This catches cases where the nearest neighbour happens to be a deleted or excluded memory.

Configurable Mood Weighting (SC_MOOD_WEIGHT_SIGNED) The v5 mood multiplier was 1 + 0.2 * abs(valence) — symmetric, meaning anger and joy memories received the same resonance boost. v6 adds SC_MOOD_WEIGHT_SIGNED=signed mode which uses 1 + 0.2 * valence instead, boosting positive memories and gently penalising negative ones in search ranking. The default preserves v5 behaviour.

Recency Boost (SC_RECENCY_HALFLIFE, --recency flag) An optional exponential decay multiplier: recency_w = exp(-ln2 / halflife * age_days). With SC_RECENCY_HALFLIFE=365, a memory from one year ago scores ~50% of an equivalent memory from today. Disabled by default (halflife=0). Exposed in the CLI as --recency and in the UI as a checkbox.

Static HTML Export (export --format html) export_vault(fmt="html") generates a self-contained single-file HTML archive using an inline Jinja2 template. The output includes a responsive card grid with mood filtering via a pure-JS filter, thumbnails, tags, and transcripts. No server required — open it in any browser, share it, or view it on mobile.

Hierarchical Tags Tags now support category:value syntax (e.g. place:paris, with:alice, mood:wistful). The DB schema, CLI, and UI accept these without any changes — they're just stored as comma-separated strings with colons. The HTML export template renders them with the colon intact.

Behaviour Changes

search_memories() now accepts recency_boost: bool parameter (default False).

delete_memory() now accepts purge_media: bool parameter (default False).

Negative query behaviour changed from hard-filter to soft-penalty — results are no longer dropped, they score lower.

doctor() now checks for jinja2 availability (needed for HTML export).

Resonance explanation string includes recency multiplier and negative penalty annotation.
